Przykład

DANE: Mamy banknot o wartości $R$ i chcemy rozmienić go na monety o nominałach $c_1,c_2,...,c_k$.
ZADANIE: Rozmienić baknot na możliwe najmniejszą liczbę monet. 

result = [0] * k; 

function find_exchange (R, curr_coins) {
if (R == 0) return 0;

var succes = -1;
var min_nr = infty;

for (let i = 1; i = k; i++) {
if (R >= c[i]) {
  var temp = find_exchange(R-c[i], curr_coins + 1);
  if (temp  min_nr) {
    min_nr = temp;
    succes = i;
  }
}
}

result[i]++;
return min_nr;
}
Rozwiązanie tego zadania budujemy od pustego zbioru monet. Potem sprawdzamy nominały od najwyższego i dorzucamy je na odpowiednią kupkę wyniku aż do momentu, gdy $R$ stanie się mniejsze od aktualnego nominału.
